WP-WebAuthn One Time Login Link Provider provides one-time login link capability as a backup for WP-WebAuthn.

== Description ==

With this plugin, when using WP-WebAuthn, users can create a one-time login link to login to the site. This is a backup login method in case WP-WebAuthn is not working or the user does not have a supported authenticator.

== Installation ==

Before installing this plugin, please make sure you have [WP-WebAuthn](https://wordpress.org/plugins/wp-webauthn/) version 1.4.0+ installed and activated.

Just download the plugin from releases and install it, then you are good to go.

You can also install [WP-WebAuthn One Time Login Link Provider](https://wordpress.org/plugins/wp-webauthn/) from WordPress plugin directory.
